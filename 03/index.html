основні властивості css для роботи з текстом

  • Властивість font-size
  • Властивість font-weight
  • Властивість font-style
  • Властивість font-family
  • Властивість color

  • Властивість text-decoration

Властивість font-size

Властивість font-size змінює розмір шрифта.

Розмір шрифту задається за допомогою значення та одиниць виміру(em, px, pt, %).

Синтаксис:

CSS

    p{
        font-size: 20px;
    }

HTML

    <p>Hello, world!</p>

Результат:

Hello, world!

Властивість font-weight

Властивість font-weight визначає насиченість шрифту.

Може приймати наступні значення:

  • normal - нормальна насиченість, те саме що й 400
  • bold - напівжирний текст, те саме що й 700
  • lighter - насиченість шрифту буде меншою, ніж у предка
  • bolder - насиченість шрифту буде більшою, ніж у предка
  • 100, 200, 300, 400, 500, 600, 700, 800, 900 - цифрові значення насиченості

Синтаксис:

CSS

    .bold{
        font-weight: bold;
    }
    .thin{
        font-weight: 100;
    }
    .max{
        font-weight: 900;
    }

HTML

    <p>
        <span class="thin">Найтонший текст </span>
        <span class="bold">Напівжирний текст </span>
        <span class="max">Найжирніший текст</span>
    </p>

Результат:

Найтонший текст Напівжирний текст Найжирніший текст

Властивість font-style

Властивість font-style дозволяє вибрати стиль зображення для шрифту.

Може приймати наступні значення:

  • normal - значення за замовчуванням
  • italic - виділяє текст курсивом

Синтаксис:

CSS

    .normal{
        font-style:normal;
    }
    .italic{
        font-style:italic;
    }

HTML

    <p>
        <span class="normal">Нормальний текст </span>
        <span class="italic">Курсивний текст</span>
    </p>

Результат:

Нормальний текст Курсивний текст

Властивість font-family

Властивість font-family використовується для вибору зображення шрифту.

Має наступні значення:

  • family-name - назва (ім'я) сімейства шрифтів, наприклад Times, Courier, Arial.
  • generic-family - базове сімейство. CSS визначає п'ять основних сімейств шрифтів: Serif, Sans-serif, Monospace, Cursive

Синтаксис:

CSS

    .tnr{
        font-family: 'Times New Roman', Times, serif;
    }

HTML

    <p>
        Roboto
        <span class="tnr">Times New Roman</span>
    </p>

Результат:

Roboto Times New Roman

Властивість color

Властивість color встановлює колір тексту.

Синтаксис:

CSS

    .red{
        color:red;
    }

HTML

    <p>
        <span class="red">Red</span>
    </p>

Результат:

Red

Властивість text-transform

Властивість text-transform .

Може приймати наступні значення:

  • uppercase - Усі символи тексту стають великими(верхній регістр)
  • lowercase - Усі символи тексту стають малими(нижній регістр)
  • capitalize - Перший символ кожного слова в реченні буде великим. Інші символи свого вигляду не змінюють.

Синтаксис:

CSS

    .upper{
        text-transform: uppercase;
    }
    .lower{
        text-transform: lowercase;
    }
    .cptlz{
        text-transform: capitalize;
    }

HTML

    <p>
        <span class="upper">uppercase</span>
        <span class="lower">LOWERCASE</span>
        <span class="cptlz">capitalize</span>
    </p>

Результат:

uppercase LOWERCASE capitalize

Властивість text-decoration

Властивість text-decoration додає оформлення тексту як його підкреслення, перекреслення чи лінії над текстом..

Може приймати наступні значення:

  • line-through - Створює перекреслений текст
  • overline - Лінія проходить над текстом
  • underline - Встановлює підкреслений текст

Синтаксис:

CSS

    .lnthrgh{
        text-decoration: line-through;
    }
    .ovrln{
        text-decoration: overline;
    }
    .undrln{
        text-decoration: underline;
    }

HTML

    <p>
        <span class="lnthrgh">line-through</span>
        <span class="ovrln">overline</span>
        <span class="undrln">underline</span>
    </p>

Результат:

line-through overline underline
