Теги заголовків

Заголовки упорядковують текст, формуючи його візуальну структуру. З використанням заголовків необхідно враховувати їх ієрархію. В заголовки не можна вкладати інші елементи
<h1> - заголовок найвищого рівня, на сторінці рекомендовано використовувати не більше 1 разу
<h2>, <h3>, <h3>, <h4>, <h5>, <h6> - заголовки нижчих рівнів, розташовані за спаданням їх можливості.

Синтаксис:

<h1>Заголовок 1-го рівня</h1>
<h2>Заголовок 2-го рівня</h2>
<h3>Заголовок 3-го рівня</h3>
<h4>Заголовок 4-го рівня</h4>
<h5>Заголовок 5-го рівня</h5>
<h6>Заголовок 6-го рівня</h6>

Результат:

Заголовок 1-го рівня

Заголовок 2-го рівня

Заголовок 3-го рівня

Заголовок 4-го рівня

Заголовок 5-го рівня
Заголовок 6-го рівня

Теги форматування тексту

<strong> - вказує на важливість тексту.
Зазвичай виділяє текст напівжирним.

Синтаксис:

Тег визначає <strong>важливість</strong> тексту.

Результат:

Тег визначає важливість тексту.

<em> - вказує на гогічне посилення тексту, уточнює змість виразу.
Зазвичай виділяє текст курсивом.

Синтаксис:

HTML - <em>головний</em> інструмент веб-розробника

Результат:

HTML - головний інструмент веб-розробника

<small> - зменшує розмір шрифту на одиницю по відношенню до звичайного тексту.

Синтаксис:

Тег <small>зменшує</small> розмір шрифту

Результат:

Тег зменшує розмір шрифту

<ins> - виділяє текст у новій версії документа, підкреслюючи його.

Синтаксис:

<ins>Версію було змінено</ins>

Результат:

Версію було змінено

<del> - перекреслює текст. Використовується для виділення тексту, видаленого з документа.

Синтаксис:

Ціна: <del>49.99</del><ins>35.99</ins>

Результат:

Ціна: 49.9935.99

<sub> - використовується для створення нижніх індексів. Зсув тексту нижче рівня рядка, зменшуючи його розмір.

Синтаксис:

H<sub>2</sub>O

Результат:

H2O

<sup> - використовується для створення ступенів. Зсув тексту вище рівня рядка, зменшуючи його розмір.

Синтаксис:

2<sup>2</sup> = 4

Результат:

22 = 4

Теги "комп`юторного коду"

<code> - cлужить виділення фрагментів програмного коду. Відображає текст моноширинним шрифтом.

Синтаксис:

<code>Math.Pow(a, 2);</code>

Результат:

Math.Pow(a, 2);

<kbd> - відзначає фрагмент як користувач, що вводиться з клавіатури. Відображає текст моноширинним шрифтом.

Синтаксис:

Комбінація копіювання <kbd>Ctrl</kbd>+<kbd>C</kbd>

Результат:

Комбінація копіювання Ctrl+C

<samp> - застосовується виділення результату, отриманого під час виконання програми.
Відображає текст моноширинним шрифтом.

Синтаксис:

2<sup>2</sup> = <samp>4</samp>

Результат:

22 = 4

<pre> - дозволяє вивести текст на екран, зберігши початкове форматування.
Пробіли та перенесення рядків при цьому не видаляються.

Синтаксис:

<pre>
if(a > b)
    max = a;
else
    max = b;
</pre>

Результат:

if(a > b)
    max = a;
else
    max = b;

Теги цитат та визначень

<dfn> - дозволяє виділити текст як визначення.

Синтаксис:

<dfn><dfn></dfn> - тег, який дозволяє виділити текст як визначення.

Результат:

<dfn> - тег, який дозволяє виділити текст як визначення.

<abbr> - застосовується для форматування абревіатур.
Браузер зазвичай підкреслюється пунктирною лінією.
Розшифровка скорочення здійснюється за допомогою атрибуту title, вона з'являється при наведенні курсору на текст.

Синтаксис:

<abbr title="HyperText Markup Language">HTML</abbr> - головний інструмент веб-розробника

Результат:

HTML - головний інструмент веб-розробника

<q> - використовується виділення коротких цитат.
Браузерами полягає у лапки.

Синтаксис:

Цицерон: <q>Кожному своє гарно.</q>.

Результат:

Цицерон:Кожному своє гарно.

<cite> - застосовується виділення цитат, назв творів, виносок інші документи.

Синтаксис:

Гра <cite>Diablo<cite> забирає нас у світ темного фентезі.

Результат:

Гра Diablo забирає нас у світ темного фентезі.

<blockquote> - виділяє цитати всередині документа, виділяючи його відступами та переносами рядків.

Синтаксис:

<blockquote>
Почніть робити те, що потрібно. Потім робіть те, що можливо. І ви раптом виявите, що робите неможливе.
</blockquote>

Результат:

Почніть робити те, що потрібно. Потім робіть те, що можливо. І ви раптом виявите, що робите неможливе.

Теги абзаців, перенесення тексту

<br> - переносить текст на наступний рядок, утворюючи розрив рядка.

Синтаксис:

Hello,<br>world!

Результат:

Hello,
world!

<hr> - використовується для розділення вмісту на веб-сторінці.
Відображається у вигляді горизонтальної лінії.

Синтаксис:

Домашнє завдання <hr> Лабораторні роботи

Результат:

Домашнє завдання
Лабораторні роботи

Теги списків

<ul> - cтворює маркований список.

Синтаксис:

<ul>
    <li>1</li>
    <li>2</li>
    <li>3</li>
</ul>

Результат:

  • 1
  • 2
  • 3

<ol> - cтворює нумерований список.

Синтаксис:

<ol>
    <li>a</li>
    <li>b</li>
    <li>c</li>
</ol>

Результат:

  1. a
  2. b
  3. c

<li> - eлемент маркованого чи нумерованого списку.

Синтаксис:

<ol>
    <li>перший елемент</li>
    <li>другий елемент</li>
    <li>третій елемент</li>
</ol>

Результат:

  1. перший елемент
  2. другий елемент
  3. третій елемент

Гіперпосилання

<a> - cтворює гіпертекстові посилання.

Синтаксис:

<a href="https://www.google.com/"> google </a>

Результат:

google

Зображення

<img> - вбудовує зображення в HTML-документ за допомогою атрибута src, значенням якого є адреса зображення, що вбудовується.

Синтаксис:

<img src="img/icon_github.png" alt="icon github">

Результат:
